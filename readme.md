# 大模型聊天应用

一个基于Spring Boot和AI大模型的实时聊天应用，支持流式响应和系统提示配置。

## 项目概述

本项目是一个现代化的聊天应用，集成了AI大模型能力，提供实时的智能对话功能。应用采用前后端分离架构，后端使用Spring Boot框架，前端使用原生HTML/CSS/JavaScript实现。

## 功能特性

### 核心功能
- **实时聊天**：与AI大模型进行实时对话
- **流式响应**：支持AI生成内容的流式展示，提供更流畅的用户体验
- **系统提示配置**：允许自定义系统提示，调整AI的行为和角色
- **跨域支持**：内置CORS配置，支持跨域请求

### 技术特性
- **Spring Boot 3**：现代化的Java后端框架
- **Spring AI**：集成AI大模型能力
- **Reactor**：支持响应式编程，实现流式输出
- **原生前端**：使用HTML5、CSS3和JavaScript构建用户界面
- **RESTful API**：提供标准化的API接口

## 快速开始

### 环境要求
- JDK 17+
- Gradle 8.0+

### 安装与运行

1. **克隆项目**

2. **配置AI模型**
   - 在 `application.properties` 中配置AI模型相关参数

3. **构建项目**
   ```bash
   ./gradlew build
   ```

4. **运行应用**
   ```bash
   ./gradlew bootRun
   ```

5. **访问应用**
   打开浏览器访问：`http://localhost:8080`

## API接口

### 1. 基本聊天接口
- **URL**: `/api/chat`
- **方法**: POST
- **请求体**:
  ```json
  {
    "message": "你好，大模型"
  }
  ```
- **响应**:
  ```
  "你好！我是大模型助手，有什么可以帮助你的吗？"
  ```

### 2. 流式聊天接口
- **URL**: `/api/chat/stream`
- **方法**: POST
- **请求体**:
  ```json
  {
    "message": "你好，大模型"
  }
  ```
- **响应**: 流式文本事件流

### 3. 带系统提示的聊天接口
- **URL**: `/api/chat/with-system`
- **方法**: POST
- **请求体**:
  ```json
  {
    "userMessage": "如何学习Java？",
    "systemPrompt": "你是一位Java编程专家，请提供详细的学习建议。"
  }
  ```
- **响应**:
  ```
  "学习Java的建议：..."
  ```
